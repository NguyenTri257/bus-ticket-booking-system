# Step-by-Step Booking Guide

## Complete Booking Process

This guide walks you through the entire booking process from start to finish.

## Step 1: Search for Trips

### Navigate to Search

1. Visit the homepage
2. Locate the trip search form (usually prominently displayed)

### Enter Search Criteria

**Required Fields:**

- **Origin**: Start typing your departure city
  - Example: "Ho Chi Minh City", "Hanoi", "Da Nang"
  - Select from autocomplete suggestions
- **Destination**: Start typing your arrival city
  - Select from autocomplete suggestions
  - Must be different from origin
- **Date**: Select travel date
  - Click calendar icon
  - Choose date (must be today or future)
  - Cannot select past dates

**Optional Fields:**

- **Passengers**: Number of travelers (default: 1)
  - Use +/- buttons to adjust
  - Affects seat availability display

### Submit Search

Click **"Search Trips"** or **"Find Buses"** button

## Step 4: Select Seats

### Interactive Seat Map

**Understanding the Map:**

- **Green seats**: Available for selection
- **Gray seats**: Already booked
- **Yellow seats**: Locked by other users (temporarily unavailable)
- **Blue seats**: Your selected seats

**Seat Types:**

- **Window seats**: Near window (marked with ü™ü)
- **Aisle seats**: Near aisle (marked with üö™)
- **VIP/Premium seats**: Extra cost, marked with ‚≠ê

### Selecting Seats

1. **Click on available (green) seat** to select
2. Seat turns blue
3. **Price updates** at bottom
4. **Click again** to deselect

**For Multiple Passengers:**

- Select 2 seats for 2 passengers
- Select 3 seats for 3 passengers, etc.
- Try to select adjacent seats if possible

### Seat Pricing

- Standard seats: Base price
- VIP seats: Base price + premium
- Window/Aisle preference: Usually same price

**Example:**

- Seat A1 (Window, Standard): 450,000 VND
- Seat A2 (Aisle, Standard): 450,000 VND
- Seat VIP1 (Window, VIP): 550,000 VND

### Important Notes

‚è∞ **10-Minute Lock**: Once selected, seats are locked for 10 minutes

- Complete booking within this time
- Timer shows remaining time
- After expiry, seats are released

### Proceed to Booking

Click **"Continue"** button at bottom

## Step 5: Enter Passenger Information

### Passenger Details Form

For **each selected seat**, provide:

**Required Information:**

- **Full Name**: Legal name (as on ID)
  - Example: "John Michael Doe"
  - Must match boarding ID

**Optional but Recommended:**

- **Phone Number**: Contact number
  - Format: +84 901 234 567
  - Used for SMS updates
- **ID/Passport Number**: Identification
  - Example: 123456789
  - May be required for boarding

### Form Validation

- ‚úÖ Green checkmark = Valid entry
- ‚ùå Red error = Invalid entry
- Fix errors before proceeding

**Common Errors:**

- Name too short (minimum 2 characters)
- Invalid phone number format
- Invalid ID number (if required)

### Contact Information

**Booking Contact** (separate from passengers):

- **Email Address**: For e-ticket delivery
  - Example: john.doe@example.com
  - Must be valid and accessible
- **Phone Number**: For SMS notifications
  - Example: +84 901 234 567

### Review Entries

Double-check all information:

- Names spelled correctly
- Phone numbers complete
- Email address correct

Click **"Continue to Summary"**

## Step 7: Complete Payment

### Choose Payment Method

Select from available options:

**1. PayOS** (Credit/Debit Cards, E-wallets)

- Visa, Mastercard, JCB
- Local e-wallets
- Most popular option

**2. Momo** (Mobile Wallet)

- Momo app required
- Scan QR code or enter phone number
- Instant confirmation

**3. ZaloPay** (Digital Wallet)

- ZaloPay app required
- Quick payment
- Cashback offers sometimes available

**4. Stripe** (International Cards)

- For international travelers
- Supports multiple currencies
- Secure processing

### Payment Process

**For Card Payment:**

1. Enter card details:
   - Card number
   - Expiry date (MM/YY)
   - CVV (3-digit code)
   - Cardholder name
2. Review amount
3. Click **"Pay Now"**
4. May require 3D Secure authentication
5. Confirm on bank app/SMS OTP

**For E-Wallet (Momo/ZaloPay):**

1. Select wallet
2. Enter phone number (if required)
3. Open app on phone
4. Confirm payment in app
5. Return to booking site

### Payment Confirmation

After successful payment:

- ‚úÖ Success message displayed
- Booking reference generated (e.g., BK20260104001)
- Redirect to booking confirmation page

### Failed Payment

If payment fails:

- ‚ùå Error message shown
- Booking remains pending
- Options:
  - Try again with same method
  - Try different payment method
  - Contact support
- ‚è∞ Remember 10-minute time limit!

## Troubleshooting

### Didn't Receive E-Ticket?

1. Check spam/junk folder
2. Wait 10 minutes (sometimes delayed)
3. Login to account ‚Üí View booking
4. Download ticket from booking details
5. Contact support if still not received

### Wrong Passenger Information?

- Contact support immediately
- Minor spelling errors usually okay
- Name must match ID for boarding
- Changes may incur fees

### Need to Cancel?

1. Login to account
2. Go to "My Bookings"
3. Select booking
4. Click "Cancel Booking"
5. Select reason
6. Confirm cancellation
7. Refund processed per policy

### Missed the Bus?

- No-show bookings are non-refundable
- Contact operator immediately
- May offer alternative arrangement (rare)
- Book new trip if needed
