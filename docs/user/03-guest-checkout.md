# Guest Checkout Guide

## What is Guest Checkout?

Guest checkout allows you to book bus tickets **without creating an account**. Perfect for:

- One-time travelers
- Users who prefer not to register
- Quick bookings without signup
- Maintaining privacy

## Benefits and Limitations

### ‚úÖ Benefits

- **No registration required**: Book immediately
- **Faster process**: Skip account creation
- **Privacy**: Minimal personal data stored
- **Email access**: Receive tickets via email
- **Booking lookup**: Access with reference + email

### ‚ö†Ô∏è Limitations

- **No booking history**: Can't view past bookings in dashboard
- **No auto-fill**: Must enter details each time
- **No trip reminders**: (unless email reminders are sent to all)
- **Manual lookup required**: Need reference number to find booking
- **No saved preferences**: Can't save notification settings

## Accessing Your Booking Later

### Booking Lookup

Since you don't have an account, use Booking Lookup feature:

1. **Navigate to Booking Lookup:**
   - Click "Booking Lookup" in menu
   - Or visit: yourdomain.com/booking-lookup

2. **Enter Required Information:**
   - **Booking Reference**: e.g., BK20260104001
   - **Contact Email**: Email used during booking

   Both fields MUST match exactly.

3. **Click "Search" or "Lookup"**

4. **View Booking Details:**
   - Trip information
   - Passenger list
   - E-ticket download
   - Payment status
   - QR code

### Security Note

For your protection:

- Both booking reference AND email required
- Prevents unauthorized access
- Only person with both can view booking

## Managing Guest Bookings

### Download E-Ticket

**From Email:**

- Open confirmation email
- Click "Download E-Ticket" link
- Or open PDF attachment
- Save or print

**From Booking Lookup:**

1. Lookup your booking (reference + email)
2. Click "Download E-Ticket" button
3. PDF downloads to device

### Cancel Guest Booking

1. **Lookup your booking** (reference + email)
2. Click **"Cancel Booking"** button
3. Select cancellation reason
4. Confirm cancellation
5. Refund processed to original payment method

**Refund Timeline:**

- Cancellation 24+ hours before: 80% refund
- Cancellation <24 hours: Subject to penalties
- Processing time: 5-7 business days

### Modify Booking

Guest bookings **cannot be modified**:

- Cannot change date
- Cannot change passenger names
- Cannot change seats

**To make changes:**

1. Cancel existing booking (subject to refund policy)
2. Make new booking with correct details

## Tips for Guest Checkout

### Before Booking

1. **Use Valid Email**: Ensure email address is correct
2. **Check Spam Settings**: Whitelist sender (noreply@yourdomain.com)
3. **Save Credentials**: Prepare to save booking reference
4. **Stable Internet**: Use reliable connection
5. **Payment Ready**: Have payment method ready

### During Booking

1. **Work Quickly**: 10-minute seat lock
2. **Double-Check Email**: Type carefully
3. **Screenshot Reference**: Immediately after payment
4. **Don't Close Browser**: Until email received

### After Booking

1. **Verify Email Receipt**: Check inbox and spam
2. **Download E-Ticket**: Save to device
3. **Print Backup**: Paper copy recommended
4. **Save Reference**: Multiple locations
5. **Add Calendar Reminder**: For travel date

## Troubleshooting Guest Bookings

### Problem: Didn't Receive Email

**Solutions:**

1. Check spam/junk folder
2. Wait 10 minutes (sometimes delayed)
3. Lookup booking online using reference + email
4. Download ticket from booking details page
5. Contact support with booking reference

### Problem: Lost Booking Reference

**Solutions:**

1. Check email confirmation (reference in subject/body)
2. Check SMS if phone was provided
3. Check payment confirmation (if paid by card)
4. Check browser history for confirmation page
5. Contact support with:
   - Contact email used
   - Travel date
   - Route (origin ‚Üí destination)
   - Approximate booking time
   - Payment amount

### Problem: Email Incorrect

If you entered wrong email:

1. **Immediately contact support** before trip date
2. Provide:
   - Booking reference (if you have it)
   - Correct email address
   - Wrong email entered
   - Payment proof
3. Support can update email and resend ticket

### Problem: Cannot Lookup Booking

**Possible Reasons:**

- Wrong booking reference format
- Wrong email address
- Typo in either field
- Booking was cancelled

**Solutions:**

- Check reference format: BKYYYYMMDDNNNNN
- Verify email exactly matches booking
- Check confirmation email for exact reference
- Contact support

## Guest vs Registered User

### When to Use Guest Checkout

Choose guest checkout if:

- ‚úÖ One-time trip
- ‚úÖ Don't want account
- ‚úÖ Value privacy
- ‚úÖ In a hurry

### When to Register

Consider registering if:

- ‚úÖ Frequent traveler
- ‚úÖ Want booking history
- ‚úÖ Want trip reminders
- ‚úÖ Prefer auto-filled forms
- ‚úÖ Want to rate trips easily

### Can I Register Later?

Yes! After guest booking:

1. Create account with same email
2. Your booking will appear in history automatically
3. Future bookings easier with account

## Converting to Registered User

### After Guest Booking

If you decide to create account:

1. **Create Account:**
   - Use **same email** as guest booking
   - Set password
   - Complete registration

2. **Benefits:**
   - Guest booking automatically linked
   - Appears in "My Bookings"
   - Can manage all bookings in one place

3. **Past Bookings:**
   - Bookings made with same email are associated
   - Can still lookup older bookings manually if different email used

## Frequently Asked Questions

### Q: Do I need to print the e-ticket?

**A:** Digital (on phone) is usually sufficient, but printing recommended as backup in case of:

- Phone battery dies
- Email app issues
- Network problems

### Q: Can I book for someone else as guest?

**A:** Yes! Enter their information as passenger, but use your email for ticket delivery. Forward ticket to them.

### Q: Is guest booking secure?

**A:** Yes, same security as registered users:

- SSL encryption
- Secure payment gateways
- Data protection

### Q: Can I get invoice for guest booking?

**A:** Payment receipt is sent via email. For formal invoice, contact support with booking reference.

### Q: What if I book multiple trips as guest?

**A:** Each booking requires separate lookup with reference + email. Consider registering for easier management.

### Q: Can I save payment details as guest?

**A:** No, guest bookings don't save payment information. Must enter each time.
